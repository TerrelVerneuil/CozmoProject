Materials Needed

  • 1x Anki Cozmo Robot
  • 1x Android Phone
  • 1x Computer with software below
  • 1x Micro USB or USB C cable to connect to the computer

Improvements

  • Image compression has improved speed of localization from original code by approximately 20s, bringing the average down from around 140 seconds to 45 seconds. This is a 32% improvement.
  • In the worst case of the original code, localization took 5 minutes. The worst case we have observed since completing changes is 1 minute.
  • When running the original code, the robot would tilt his head up while collecting pictures, but not while localizing. We made it so he tilts his head while localizing, so there is less variance between images taken from the same angle.
  • Threshold modification has increased certainty from an average of ~40% to an average of ~80%
  • The robot now sings "Ode to Joy" upon localizing
