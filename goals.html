Introduction:

By using the Anki Cozmo robotics platform, We will simulate the Kidnapped Robot Problem, using an implementation of Monte Carlo Localization. By using the Anki Cozmo Robot it should figure out where it is, and based on the culmination of its rotation and location data, the robot should be able to familiarize itself with its environment, and figure out where it is after being “Kidnapped”.

Project Goals

  • Understand previous implementations of the project.
  • Implement Monte Carlo Localization based on these references.
  • Simulate the Kidnapped Robot Problem.
  • Use an Anki Cozmo Robot to demonstrate Monte Carlo Localization.
  • Collect rotation and location data to help the robot familiarize itself with its environment.
  • Enable the robot to determine its location after being “kidnapped.”
  • Improve upon previous implementations of the Kidnapped Robot Problem.
