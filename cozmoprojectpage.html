Anki Cozmo Localization


Gettysburg College Computer Science

CS371 Spring 2023

Authors: Jake Dressler, Mike Dolan, Connor Whitlow, Joe Hillesland, Terrel Verneuil, Giorgi


Introduction:

By using the Anki Cozmo robotics platform, We will simulate the Kidnapped Robot Problem, using an implementation of Monte Carlo Localization. By using the Anki Cozmo Robot it should figure out where it is, and based on the culmination of its rotation and location data, the robot should be able to familiarize itself with its environment, and figure out where it is after being “Kidnapped”.
